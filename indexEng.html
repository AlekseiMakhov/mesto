Why?

The project is developed to show the advantage of the quick sorting algorithm. We have 2 parameters as result of execution - count of cycles, duration in milliseconds.

What?

There is realized 2 ways of the source array input - manual and auto (for lazy people, input of several thousands numbers requires a lot of time, patience and persistence).

What to do?

- In manual input mode is offered to input number between 0 and 9999. Then press 'Enter' to push the number to source array.

- In auto input mode is offered to input an array items count between 1 and 99999. Then click 'Generate' to get the automatically generated source array of random numbers from 0 to 9999.

- Then click 'Sort' for start the algorithms. I have to warn you: a large count of array items greatly increases the simple sorting duration. Computer is working, please wait...

- Button 'Reset' is resetting all values and fields.
